# NeuroVision AI

MRI-Based Brain Tumor and Alzheimer's Disease Detection System

## Overview

NeuroVision AI is a deep learning-based medical imaging application that assists in the detection of:

- Brain Tumors
- Alzheimer's Disease

The system uses pretrained ResNet18 models and Grad-CAM visualization to provide prediction results along with interpretable heatmaps.

## Features

- Brain Tumor Detection
- Alzheimer's Disease Detection
- Grad-CAM Explainability
- MRI Image Upload
- Interactive Streamlit Interface
- Medical Disclaimer and User Consent

## Technologies Used

- Python
- PyTorch
- Streamlit
- OpenCV
- NumPy
- Pillow
- Grad-CAM
